Controls For Sprint2:
O/P: Change Displayed Enemy
WASD: Change Mario Position
U/I: Change Displayed Item
T/Y: Change Displayed Block
X/M: Change Mario Big/Small
J: Make Mario Fire Flower State
Q: Quit Game
R: Reset Game
Z/N: Throw Fireball
